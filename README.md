## About This Repository

This repository contains **6 Jupyter Notebooks** written entirely by hand,
covering the core building blocks of Python programming. Every notebook
includes clear markdown explanations alongside working code cells — written
the way a student actually thinks through a topic, not just copied from
documentation.

These notebooks were built as a personal learning reference and are shared
here as a portfolio of foundational Python knowledge.

## Notebook Breakdown

### 01 — Python Basics and Intermediate
`93 code cells · 17 explanation cells`

The starting point. Covers how Python works from the ground up including
data types, variables, keywords, identifiers, user input, type conversion
(both implicit and explicit), literals, dynamic and static typing, and
dynamic and static binding. A strong foundation before moving to anything else.

---

### 02 — Operators, Conditions and Loops
`137 code cells · 37 explanation cells`

The largest notebook in the collection. Covers all operator types in Python —
arithmetic, relational, logical, bitwise, assignment, and membership operators
— with clear examples for each. Then moves into conditional logic with if,
else, and elif statements including real-world examples like a login system
and a menu-driven program. Also covers Python modules, the difference between
built-in and external modules, and how to import and use them.

---

### 03 — Lists and List Functions
`71 code cells · 15 explanation cells`

A thorough walkthrough of Python lists. Covers how Python stores objects in
memory, creating lists, accessing items by index, adding items using append,
insert and extend, editing and deleting items, list operations, and all major
built-in list functions. Also includes list comprehension with detailed
explanations of why it is useful, how to traverse a list, and the zip()
function.

---

### 04 — Tuples and Tuple Functions
`40 code cells · 12 explanation cells`

Covers Python tuples from creation to use. Explains the key difference between
tuples and lists — immutability — and when to use each one. Includes accessing
items, attempting edits, tuple operations, all built-in tuple functions, and a
clear comparison table between tuples and lists. Also covers special syntax
cases that beginners commonly get wrong.

---

### 05 — Sets and Set Functions
`62 code cells · 14 explanation cells`

Covers the set data type in detail. Explains what sets are, when to use them,
and a common beginner mistake with curly braces that creates a dictionary
instead of a set. Covers editing and deleting items, all set operations, major
built-in set functions, the four update functions, frozen sets and which
functions work on them and which do not, and set comprehension.

---

### 06 — Dictionaries and Dictionary Functions
`42 code cells · 11 explanation cells`

Covers Python dictionaries from scratch using real-world analogies like a
phone book and a real dictionary. Includes creating 1D, homogeneous,
heterogeneous, and nested dictionaries, accessing and editing key-value pairs,
removing items with pop and del, dictionary operations, the three most
important dictionary functions — keys(), values(), and items() — and
dictionary comprehension including nested comprehension.

## 🛠️ How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/Hosen-GIT/python-fundamentals.git
   ```

2. Open any notebook in Jupyter
   ```bash
   jupyter notebook
   ```

3. Navigate to the notebook you want and run cells with **Shift + Enter**

> No extra libraries required — everything uses core Python only.
